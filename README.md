# LabBridge 🚀

LabBridge is a real-time cross-device text and code sharing platform designed for students and computer labs.

## Features
- QR-based session connection
- Real-time text transfer
- Cross-device communication
- Session-based architecture
- No login required

## Tech Stack
- Flask
- Flask-SocketIO
- HTML/CSS/JavaScript
- QRCode

## How It Works
1. Open website on computer
2. Scan QR using phone
3. Paste/send text
4. Text instantly appears on computer

## Future Features
- File transfer
- Image sharing
- Video sharing
- Syntax highlighting
- Clipboard sync
- Session expiry
